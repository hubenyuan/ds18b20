# ds18b20
## 介绍
本项目主要是树莓派上通过一线协议(1-Wire)连接DS18B20，采用网络socket进行客户端和服务器的连接。
客户端负责实现定时上报数据的功能，服务器则用来接受数据并把数据永久保存到数据库中。
